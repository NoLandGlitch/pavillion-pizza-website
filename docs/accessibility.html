Accessibility Statement

Pavillion Ristorante & Pizzeria is committed to making this website usable by as many people as possible, including people using assistive technology. This site is a work in progress, and we welcome feedback. If you encounter an accessibility barrier anywhere on this site, please let us know so we can address it.
