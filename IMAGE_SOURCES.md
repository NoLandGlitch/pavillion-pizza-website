# Image Sources — Pavillion Ristorante & Pizzeria Redesign

All images below are **presentation stock photography only** — none are
actual photographs of Pavillion Ristorante & Pizzeria, its staff, its
kitchen, or its dining room. Do not caption or alt-text any of these as
if they depict the real restaurant. Real photography (owner/kitchen/
dining room/food actually made at Pavillion) should replace these as
soon as it's available — see `CONTENT_NEEDED.md`.

**Update:** the static site build (`site/assets/images/`) now contains
actual downloaded copies of every image below — none are hotlinked, all
were fetched from Unsplash's CDN and saved locally under the filenames
listed here. If this content is later moved into a Wix build (see
`WIX_SETUP.md`), re-upload these same local files into the Wix Media
Manager rather than re-fetching from Unsplash.

**Do not hotlink.** Every image must be downloaded from its source page
before use — never reference `images.unsplash.com` URLs directly from
live page code.

**Licensing note:** Unsplash License and Pexels License both permit free
commercial use with no attribution required. Attribution is listed below
anyway as a courtesy and for your own records — verify the license badge
on the individual photo's page at download time in case a contributor
has since changed terms.
